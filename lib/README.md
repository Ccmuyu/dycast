一个基于**QT**的抖音弹幕获取工具

## 简介

**说明：** 用户只需输入房间号，即可实时获取直播间弹幕信息；同时，可通过`ws`同步转发到自己的后端服务

**技术栈：** `python-3.11`、`PySide6`、`Websocket`、`protobuf`、`···`

**版本：** `0.9.2`

## 功能介绍

- `0.9.2`
  - 弹幕获取
  - 弹幕展示
  - 主播信息展示
  - 弹幕转发
  - 连接计时
  - 连接状态提示
